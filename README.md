FasterMate

– Developed an innovative web platform that empowers individuals to list their vehicles for rent, while providing
users with a seamless experience to search for and lease vehicles from fellow community members.

– Used Handlebars for dynamic front-end and integrated MongoDB for efficient user and vehicle data storage.

– Enhanced user experience by integrating Google Maps API for dynamic car location display.

– Technology used: Node.js, Express.js, MongoDB, HandleBars, Google Maps API
